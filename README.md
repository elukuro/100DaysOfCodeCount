### Tools to preview montly data of tweet with #100daysofcode hashtag to preview their progress

### how to setup:
```
git clone
```

```
cd static
```

```
npm install 
```

###  Run in development mode 
```
npm run serve 
```


## Build for production mode 
```
npm run build 
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
