# 100DaysOfCodeCount

### tools to preview montly tweet of user that use #100daysofcode hashtag to preview their journey

### how to setup:
```
git clone the project
```

```
cd frontend
```

```
npm install 
```

###  Run in development mode 
```
npm run serve 
```


## Build for production mode 
```
npm run build 
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
